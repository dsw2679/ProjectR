## 1. 무기 사운드

### 1.1 발사음

| 무기 | 사운드 특성 | 재생 조건 |
| --- | --- | --- |
| 돌격소총 | 연속적이고 빠른 총성. 중~고음역대 기계음 | 발사 입력마다 1회 재생 |
| 유탄발사기 | 묵직한 단발 발포음 + 발사 후 탄두 바람음 | 발사 시 1회 재생 |
| 볼트액션 | 날카로운 강타음 + 볼트 후퇴음(별도 레이어) | 발사 시 발포음, 볼트 모션 시 볼트음 |

- 사운드 에셋은 무기 데이터 테이블에서 사운드 에셋 관리
- 거리 감쇠(Attenuation): 발사음은 최대 3000 cm에서 자연 감쇠

### 1.2 재장전음

| 무기 | 사운드 구성 |
| --- | --- |
| 돌격소총 | 탄창 분리음 + 탄창 삽입음 + 슬라이드 복귀음 (3레이어) |
| 유탄발사기 | 장전구 개방음 + 유탄 삽입음 + 장전구 닫힘음 |
| 볼트액션 | 볼트 개방음 + 탄환 삽입음 + 볼트 닫힘음 |

- 재장전 모션 타이밍에 레이어별 재생 (레이어별 재생 구간은 모션 작업 시 확정)

### 1.3 탄창 소진 알림음

- 탄창 잔탄 0 도달 시 짧은 알림음 재생 (빈 클릭음)
- 자동 재장전 시작 직전 재생

### 1.4 탄 피격음 (표면 재질별 분기)

피격 또는 투사체가 표면에 충돌 시 표면 재질 태그 기반으로 사운드 분기.

| 표면 재질 | 사운드 |
| --- | --- |
| 육체 (적 히트) | 타격음 (육질감) |
| 금속 | 금속 충돌음 |
| 콘크리트 / 석재 | 분진 충격음 |
| 나무 | 목재 분쇄음 |
| 기본 (분류 불명) | 일반 충격음 |

## 2. 전투 사운드

### 2.1 적 피격음 / 그로기 발동음

| 이벤트 | 사운드 |
| --- | --- |
| 일반 적 피격 | 적 타입별 피격 반응음 |
| 약점 적중 | 강조된 충격음 (고음역 히트 효과) |
| 그로기 진입 | 적 전용 그로기 진입 사운드 (단발 강타감) |

### 2.2 폭발음

- 투사체 폭발 시 폭발 사운드 재생
- 거리 감쇠 적용: 3000 cm 이상에서 낮은 두께감만 유지
- 공간감 있는 저음 잔향 포함

### 2.3 보스 페이즈 전환 연출음

- 페이즈 전환 시 극적인 전환 효과음 + 짧은 공백 후 새 페이즈 BGM 시작
- 전환 연출 타임라인에 동기화

### 2.4 플레이어 피격 / 사망 사운드

| 이벤트 | 사운드 |
| --- | --- |
| 피격 | 짧은 충격음 + 플레이어 반응음 |
| 체력 위험(30% 이하) | 심박 사운드 반복 재생 (2D, 비공간음) |
| 다운 상태 진입 | 낮고 둔탁한 충격음 + 이명 효과 |
| 완전 사망 | 사망 사운드 |
| 소생 완료 | 소생 완료 효과음 |

## 4. 환경 / BGM

### 4.1 앰비언트 사운드

- 구역별 독립 앰비언트 사운드 에셋 할당
- 카테고리별 볼륨 독립 조절 지원

| 구역 유형 | 앰비언트 특성 |
| --- | --- |
| 거점 | 조용하고 안정적인 환경음 |
| 전투 구역 (탐색 중) | 긴장감 있는 효과음 |
| 보스 아레나 (전투 전) | 정적 또는 불길한 저음 루프 |

### 4.2 BGM 전환 규칙

| 상태 전이 | BGM 처리 |
| --- | --- |
| 거점 입장 | 거점 BGM 페이드인 |
| 전투 구역 진입 (탐색) | 탐색용 BGM 전환 |
| 전투 개시 (적 감지) | 전투 BGM 크로스페이드 전환 |
| 전투 종료 (적 전멸) | 탐색 BGM 또는 앰비언트로 복귀 |
| 보스전 시작 | 보스 Phase A BGM 전환 |
| 보스 페이즈 전환 | 해당 페이즈 BGM 트랙 크로스페이드 |
| 보스 처치 | 승리 연출 사운드 + BGM 종료 또는 거점 BGM 전환 |

### 4.3 BGM 구성 상세

| BGM | 구간 | 설명 |
| --- | --- | --- |
| 거점 BGM | 거점 체류 중 | 안정감 있는 환경 음악 |
| 탐색 BGM | 전투 구역 탐색 중 | 긴장감과 모험감 혼재 |
| 전투 BGM | 일반 전투 중 | 빠른 리듬, 전투 집중감 |
| 보스 Phase A BGM | 보스 100~75% | 중간 강도. 위협감 |
| 보스 Phase B BGM | 보스 75~33% | 강도 상승. 긴박감 |
| 보스 Phase C BGM | 보스 33% 이하 | 최고 강도. 절박감 |
