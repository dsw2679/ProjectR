## 1. 전투 HUD

### 1.1 체력 바 (3중 레이어)

| 레이어 | 색상/표현 | 역할 |
| --- | --- | --- |
| 현재 체력 바 | 진한 색상 (전경) | 실시간 체력 표시 |
| 회복 가능 체력 바 | 중간 밝기 색상 | 회복 가능한 게이지 표시 |
| 지연 체력 바 | 연한 색상 (배경) | 피해 후 느리게 감소하는 시각 피드백 |

- 위치: 화면 좌측 하단
- 체력 30% 이하 시 바 색상 위험 색으로 변경 및 맥동 효과 적용

### 1.2 스태미너 바

- 체력 바 하단에 배치
- 스태미너 소모 시 즉시 감소, 회복 딜레이 후 자동 충전 표시
- 고갈 상태 시 시각적 강조 (깜박임 등)

### 1.3 탄약 표시

- 화면 우측 하단
- 현재 탄창 잔탄 / 보유 탄약 형식 표시 (예: **24 / 150**)
- 잔탄 5발 이하 시 경고 색상으로 변경

### 1.4 Mod 게이지

- 탄약 표시 근처에 배치
- 게이지 충전 진행률 표시
- 100% 충전 완료 시 발동 가능 알림 UI 및 사운드 재생

### 1.5 크로스헤어

- 화면 중앙에 표시
- ADS 상태 진입 시 크로스헤어 형태 전환
- 볼트액션 스코프 진입 시 크로스헤어 숨김 (스코프 오버레이로 대체)
- 사격 시 확산 애니메이션 적용

### 1.6 상태이상 아이콘

- 체력 바 근처에 현재 적용 중인 상태이상 아이콘 표시
- 아이콘에 남은 지속 시간 타이머 표시
- 임시 상태이상과 저주(Blight) 계열 아이콘 시각적으로 구분

## 2. 적/보스 HUD

### 2.1 일반 적 체력 바

- 적 피격 시 적 머리 위에 체력 바 표시
- 일정 시간(3 초) 이후 자동 숨김

### 2.2 보스 체력 바

- 화면 상단 중앙에 항상 표시 (보스 아레나 진입 시 활성화)
- 보스 이름 및 현재/최대 HP 표시
- 페이즈 임계값 위치에 마커 표시 (75%, 33% 위치)

### 2.3 보스 그로기 게이지

- 보스 체력 바 하단에 별도 게이지로 표시
- 그로기 상태 진입 시 게이지 시각적 강조 및 그로기 중 타이머 표시

### 2.4 페이즈 UI

- 보스 페이즈 전환 시 화면 중앙에 페이즈 알림 텍스트 표시 (예: "Phase B")
- 전환 연출 중 화면 가장자리 이펙트 적용

### 2.5 전멸기 캐스팅 바

- 보스가 전멸기 캐스팅 시작 시 화면 상단에 경고 캐스팅 바 표시
- 바 색상: 위험/경고 색 (적색 계열)
- 캐스팅 취소 시 바 즉시 사라짐

### 2.6 피격 숫자 색상

| 색상 | 의미 |
| --- | --- |
| 흰색 | 일반 피해 |
| 빨간색 (굵게) | 약점/치명 피해 |

## 3. 멀티플레이 HUD

### 3.1 파티원 체력 표시

- 화면 좌측에 파티원 목록 표시
- 각 파티원: 이름, 체력 바, 다운 여부 표시

### 3.2 다운 카운트다운

- 파티원이 다운 상태 진입 시 파티원 목록에 카운트다운(30초) 표시
- 카운트다운 만료 전 소생 시 카운트다운 사라짐

### 3.3 소생 바

- 소생자 입장에서: 소생 진행 바가 화면 하단 또는 대상 위에 표시
- 다운 플레이어 입장에서: 소생자 방향으로 소생 진행 바 표시

### 3.4 위치 마커

- 파티원 위치를 화면 가장자리에서 방향 마커로 표시 (시야 밖일 경우)
- 다운 상태 파티원은 마커 색상을 다르게 표시 (예: 주황/빨간)

## 4. 인벤토리/장비 UI

### 4.1 인벤토리 화면

- I 키로 진입. 전체 화면 오버레이 형태
- 카테고리 탭: 무기 / Mod / 소비 / 재료
- 각 슬롯에 아이템 썸네일, 이름, 수량 표시
- 우측 패널: 선택 아이템 상세 정보 (이름, 효과 설명, 장착/사용/판매 버튼)

### 4.2 무기/Mod 장착 화면

- 장착 화면에서 무기와 Mod를 드래그하거나 선택하여 장착
- 현재 장착 중인 무기/Mod 강조 표시
- 무기 슬롯에 Mod 슬롯 하위 표시

### 4.3 퀵슬롯 HUD

- 화면 하단에 소비 아이템 퀵슬롯 4칸 항상 표시 (2~5번 키 대응)
- 용의 심장 슬롯(1번)은 별도 강조 표시
- 아이템 수량 1 이하 시 경고 색상 표시

### 4.4 상세 정보 패널

- 아이템 선택 시 우측 패널에 표시:
  - 아이템 이름 및 카테고리
  - 썸네일 이미지
  - 효과 설명 텍스트
  - 현재 장착 여부
  - 장착/사용/분해 버튼 (카테고리에 따라 버튼 구성 변화)

## 5. 상호작용 UI

### 5.1 프롬프트 표시

- 감지 범위 내 상호작용 가능 오브젝트 감지 시 오브젝트 위에 프롬프트 표시
- 프롬프트 구성: 오브젝트 이름 + 입력 키 아이콘

### 5.2 즉발/홀드 구분 연출

| 상호작용 유형 | UI 표현 |
| --- | --- |
| 즉발 (단순 키 입력) | 키 아이콘만 표시 |
| 홀드 (소생 등) | 키 아이콘 + 원형 게이지 진행 바 표시 |

## 7. 메뉴 시스템

### 7.1 메인 메뉴

| 항목 | 설명 |
| --- | --- |
| 새 게임 | 새로운 세이브 슬롯 생성 후 게임 시작 |
| 계속하기 | 마지막 세이브 데이터에서 이어하기 |
| 슬롯 선택 | 세이브 슬롯 목록 표시 후 선택 |
| 설정 | 설정 메뉴 진입 |
| 종료 | 게임 종료 |

### 7.2 인게임 일시정지 (싱글)

- Tab 키로 일시정지 메뉴 진입
- 항목: 계속하기, 설정, 메인 메뉴로 이동, 게임 종료

### 7.3 인게임 오버레이 (멀티)

- 멀티 진행 중 일시정지 불가. Tab 키로 오버레이 메뉴만 표시
- 항목: 설정, 세션 퇴장, 게임 종료

### 7.4 설정 메뉴

| 카테고리 | 설정 항목 |
| --- | --- |
| 그래픽 | 해상도, 품질 프리셋, 프레임 제한 |
| 사운드 | 마스터/SFX/BGM/UI 볼륨 개별 조절 |
| 키 바인딩 | 각 액션별 키 재설정 |
| 기타 | 자막 여부 등 |

## 8. 핑/마커 시스템

### 8.1 핑 발신

- G 키 입력으로 현재 조준 지점 또는 오브젝트에 핑 발신
- 서버에서 핑 위치 및 대상 오브젝트 정보 검증 후 전파

### 8.2 핑 종류

| 종류 | 표시 | 설명 |
| --- | --- | --- |
| 위치 핑 | 지점 마커 | 특정 위치 지시 |
| 적 마킹 | 적 강조 마커 | 적 오브젝트 핑 시 적 마킹 표시 |
| 위험 알림 | 경고 마커 | 위험 지역 또는 상황 알림 |

### 8.3 핑 지속 시간 및 제거

- 핑 기본 지속 시간: 5 초
- 지속 시간 만료 시 마커 자동 제거
- 마커 제거는 모든 플레이어에게 동시 반영

## 9. 대미지 텍스트

### 9.1 표시 방식

- 피해를 가한 지점 근처에 떠오르는 숫자 텍스트로 표시
- 복수 피해가 짧은 시간 내 발생 시 누적 합산하여 표시 (스팸 방지)

### 9.2 최대 팝업 수 제한

- 동시에 표시되는 대미지 텍스트 최대 수: 10 개
- 초과 시 오래된 텍스트부터 제거

### 9.3 타입별 색상

| 색상 | 피해 유형 |
| --- | --- |
| 흰색 | 일반 피해 |
| 빨간색 | 약점/치명 피해 |
