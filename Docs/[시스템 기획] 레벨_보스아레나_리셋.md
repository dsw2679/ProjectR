## 1. 보스 아레나 시스템

### 1.1 아레나 진입 흐름

```mermaid
flowchart TD
    A[플레이어 아레나 입구 진입] --> B[준비 대기 상태 UI 표시]
    B --> C{멀티플레이?}
    C -->|No| D[즉시 진입 가능]
    C -->|Yes| E[파티원 전원 준비 완료 입력 대기]
    E --> F{전원 준비 완료?}
    F -->|No| E
    F -->|Yes| D
    D --> G[아레나 진입 확정]
    G --> H[최초 진입 여부 확인]
    H -->|최초| I[보스 진입 시네마틱 재생 TBD]
    H -->|재진입| J[직접 전투 시작]
    I --> J
    J --> K[보스 스폰 및 퇴로 장벽 활성화]
```

### 1.2 아레나 진입 규칙

- **싱글**: 진입 즉시 확정
- **멀티**: 아레나 입구에 준비 상호작용 존재. 파티원 전원이 준비 입력 시 동시 진입
