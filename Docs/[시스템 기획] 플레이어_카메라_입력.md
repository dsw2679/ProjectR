## 1. 카메라 시스템

### 1.1 기본 시점

- 3인칭(TPS) 어깨 너머 시점. 기본적으로 오른쪽 어깨 기준

| 항목 | 값 (기본값) |
| --- | --- |
| 카메라 암 길이 | 300 cm |
| 어깨 오프셋 (X, Y, Z) | (0, 60, 80) cm |
| 카메라 FOV (기본) | 90 도 |

### 1.2 카메라 충돌 처리

- 카메라와 캐릭터 사이에 장애물(벽, 기둥 등) 감지 시 카메라를 장애물 앞으로 즉시 당김
- 장애물 제거 시 지정 속도(기본값 5 cm/s)로 원래 위치로 복귀

### 1.3 ADS 전환

- ADS 입력 시 카메라 암 길이 단축 및 어깨 오프셋 보정으로 조준 시점 연출
- 모든 무기가 ADS FOV 값을 공유
- 볼트액션은 조준 상태에서 줌인 입력(Z키)으로 스코프 모드 진입 (TBD)

| 무기 | ADS FOV |
| --- | --- |
| 공통 | 75 도 |
| 볼트액션 스코프 | 40 도 |

- FOV 전환은 보간(Interpolation)으로 처리. 전환 속도: 0.15 초

## 2. 조준(ADS) 시스템

### 2.1 ADS 모드 진입/해제

- 조준 입력(기본: 마우스 우클릭) 유지 중 ADS 상태 유지
- 입력 해제 시 즉시 ADS 해제

| ADS 상태 | 이동 속도 | FOV |
| --- | --- | --- |
| 비활성 | 기본 속도 | 90 도 |
| 활성 | 200 cm/s | 무기별 ADS FOV |

### 2.2 ADS 중 이동 속도 감소

- ADS 진입 즉시 이동 속도를 ADS 이동 속도로 덮어씀
- 달리기 도중 ADS 입력 시 달리기 해제 후 ADS 적용

### 2.3 크로스헤어 / 스코프 오버레이

- 볼트액션 ADS 진입 시 비네팅(스코프) 오버레이 UI 활성화
  - 스코프 진입: 화면 대부분을 스코프 렌즈 이미지로 대체, 외부 화면은 검게 처리
  - 스코프 FOV: 40 도
  - 스코프 종료: 줌인 입력 해제 또는 ADS 해제
- 돌격소총, 유탄발사기의 경우 ADS 전용 크로스헤어 형태로 전환 (TBD)

## 3. 상호작용 시스템

### 3.1 감지 방식

- 플레이어 위치 기준 구형 탐지로 상호작용 가능 오브젝트 감지
- 감지 반경: 200 cm
- 0.1 초 간격으로 탐지 처리 (성능 최적화)
- 복수 오브젝트 탐지 시 가장 가까운 오브젝트 우선

### 3.2 상호작용 종류

| 상호작용 유형 | 방식 | 처리 결과 |
| --- | --- | --- |
| 체크포인트 활성화 | 단순 키 입력 | 체크포인트 활성화 및 동기화 |
| 포탈 진입 | 단순 키 입력 | 지역 선택 UI 표시 |
| 아이템 픽업 | 단순 키 입력 (E키) | 인벤토리 또는 자원에 추가 |
| 소생 | 홀드 (4 초) | 소생 진행 바 표시 후 소생 완료 |
| NPC 대화 | 단순 키 입력 | 대화/거래 UI 표시 |

### 3.3 UI 프롬프트

- 감지 범위 내 상호작용 가능 오브젝트 존재 시 해당 오브젝트 위에 상호작용 키 프롬프트 표시
- 복수 오브젝트 중 가장 우선 대상만 프롬프트 표시
- 프롬프트에는 오브젝트 이름 및 입력 키 아이콘 포함

## 4. 입력 시스템

### 4.1 지원 입력 장치

- **키보드+마우스** 지원 (확정)
- 게임패드: 미지원

### 4.2 기본 키 바인딩 (키보드+마우스)

| 액션 | 기본 키 |
| --- | --- |
| 이동 | WASD |
| 피하기(구르기) | Space |
| 앉기 | L Alt |
| 달리기 | L Shift |
| 능력 1 | Q |
| 능력 2 | C |
| 근접공격/사격 | 마우스 좌클릭 |
| 조준(ADS) | 마우스 우클릭 |
| 재장전 | R |
| 무기 모드 변경 | F |
| 줌인 (볼트액션 스코프) | Z |
| 무기 전환 | 마우스 휠 / X |
| 용의 심장 | 1 |
| 아이템 사용 1~4 | 2~5 |
| 핑 | G |
| 상호작용 | E |
| 인벤토리 | I |
| 메뉴 | Tab |

### 4.3 입력 버퍼링

- 짧은 연계 입력을 위해 입력 버퍼 구간 적용 (기본 0.15 초)
- 구르기는 입력 우선순위 최상위 — 다른 모션 진행 중에도 버퍼에 즉시 저장
- 구르기 모션 중 사격 입력 시 버퍼에 저장, 구르기 종료 후 사격 자동 실행
