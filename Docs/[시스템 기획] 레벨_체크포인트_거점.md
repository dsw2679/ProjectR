## 1. 체크포인트 시스템

### 1.1 체크포인트 상태 정의

레벨에 배치된 체크포인트 오브젝트와 상호작용하여 활성화한다.

| 상태 | 설명 |
| --- | --- |
| 비활성 | 초기 상태. 아직 활성화되지 않은 체크포인트 |
| 활성 | 플레이어가 상호작용하여 활성화. 부활 지점으로 등록됨 |

- 활성화 방식: E키 단순 입력 (홀드 불필요)
- 활성화 시 이펙트 및 사운드 재생

### 1.2 체크포인트 추적

- 게임 상태에서 마지막 활성 체크포인트 ID 유지
- 복수 체크포인트 존재 시 가장 최근 활성화된 1개만 유효 부활 지점으로 유지
- 이전 체크포인트는 활성 표시는 유지하되 부활 지점에서 제외

### 1.3 멀티플레이 체크포인트 활성화 규칙

- **생존 중인 파티원 전원**이 체크포인트 근처에서 **동시에 상호작용**해야 활성화
- 활성화 시점에 다운 또는 사망 상태인 파티원은 해당 체크포인트에서 **즉시 자동 부활**
- 전원 상호작용 전 이탈하거나 피격으로 조작이 중단되면 활성화 취소

## 2. 부활 및 리셋 규칙

### 2.1 체크포인트 부활 시 처리

- 부활 체력: 최대 체력의 50%
- 부활 위치: 체크포인트 주변 지정 스폰 위치

### 2.2 체크포인트 부활 시 리셋 항목

| 항목 | 처리 |
| --- | --- |
| 보스 HP | 리셋 (최대 HP로 복구) |
| 일반 몬스터 | **리스폰** (체크포인트 부활 시 재소환) |
| 보유 탄약 | **유지** (부활 시 탄약 페널티 없음) |
| 보유 소비 아이템 | **유지** |
| 체크포인트 활성 상태 | 유지 |

### 2.3 멀티플레이 파티 동시 부활

- 체크포인트 활성화 시점에 사망 또는 다운 상태인 파티원은 해당 체크포인트에서 즉시 부활
- 체크포인트 이후 사망한 파티원은 전멸 발생 시에만 체크포인트에서 동시 부활
- 전멸 후 파티 전체가 마지막 활성 체크포인트에서 동시 부활

### 2.4 레벨 재진입 시 전체 리셋

| 항목 | 처리 |
| --- | --- |
| 몬스터 | 전체 리스폰 |
| 아이템 드롭 | 초기 배치 복구 |
| 보스 HP | 최대 HP 초기화 |
| 체크포인트 | 세이브 데이터 기반 상태 유지 |
