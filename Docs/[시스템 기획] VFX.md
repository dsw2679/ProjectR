## 1. 전투 이펙트

### 1.1 총구 화염(Muzzle Flash)

| 항목 | 내용 |
| --- | --- |
| 재생 위치 | 총구 위치 |
| 지속 시간 | 0.05 초 (단발 버스트 이펙트) |
| 멀티플레이 | 로컬 플레이어 화면 재생 + 다른 플레이어 화면에도 재생 |

| 무기 | 화염 특성 |
| --- | --- |
| 돌격소총 | 소형, 빠른 섬광. 연사 빈도에 맞게 연속 재생 |
| 유탄발사기 | 중형, 팽창하는 화염구 + 연기 |
| 볼트액션 | 대형, 강한 섬광 + 연기 꼬리 |

### 1.2 탄환 궤적(Tracer) 및 탄착 이펙트

| 항목 | 내용 |
| --- | --- |
| 탄환 궤적 | 히트스캔 발사 시 총구 ~ 탄착 지점 라인 이펙트. 0.05 초 잔상 후 소멸 |
| 탄착 위치 | 히트스캔 충돌 지점 또는 투사체 충돌 지점 |
| 표면 재질 분기 | 표면 재질 태그 기반으로 탄착 이펙트 에셋 선택 |

| 표면 재질 | 탄착 이펙트 |
| --- | --- |
| 육체 | 혈흔 파티클 + 작은 혈액 스플래시 |
| 금속 | 스파크 파티클 |
| 콘크리트 / 석재 | 먼지 + 파편 파티클 |
| 나무 | 목재 파편 + 먼지 파티클 |
| 기본 | 소형 먼지 파티클 |

### 1.3 투사체 이펙트

| 이펙트 | 내용 |
| --- | --- |
| 투사체 비행 이펙트 | 유탄발사기 투사체 비행 중 후미 연기 트레일 |
| 폭발 이펙트 | 충돌 지점에 팽창하는 폭발 파티클 + 빛 섬광 + 충격파 링 |
| 장판 지속 이펙트 | 장판 활성 구역에 지속 파티클(불꽃, 연기, 발광 등) 루프 재생 |

### 1.4 피격 이펙트

- 적 피격 시 탄착 이펙트 + 피격 임팩트 이펙트 재생
- 약점 피격 시 추가 발광 이펙트 재생

## 2. 상태 이펙트

### 2.1 그로기 상태 이펙트 (적)

| 항목 | 내용 |
| --- | --- |
| 진입 이펙트 | 그로기 진입 시 적 색상 변조 |
| 지속 이펙트 | 그로기 중 적 캐릭터에 지속 약화 파티클 (파편, 전기, 연기 등 테마에 따라) |
| 종료 이펙트 | 그로기 종료 시 복귀 이펙트 (짧은 발광) |

### 2.2 상태이상 이펙트

| 상태이상 | 이펙트 |
| --- | --- |
| 화상(Burn) | 적/플레이어 캐릭터에 불꽃 파티클 루프 재생 |
| 출혈(Bleed) | 붉은 파티클 루프 재생 |
| 부식(Corrode) | 녹색 계열 부식 파티클 루프 재생 |
| 저주(Blight) | 어두운 보라색 또는 검은 계열 파티클 루프 재생 |

### 2.3 다운 상태 / 소생 이펙트

| 이벤트 | 이펙트 |
| --- | --- |
| 다운 상태 진입 | 플레이어 주변 미세 파티클(에너지 감소 표현) 루프 재생 |
| 소생 진행 중 | 소생자와 다운자 사이 연결 에너지 빔 이펙트 |
| 소생 완료 | 소생 완료 빛 폭발 이펙트 |

## 3. 환경 및 연출 이펙트

### 3.1 보스 페이즈 전환 연출 이펙트

| 단계 | 이펙트 |
| --- | --- |
| 전환 시작 | 보스 전신 발광 + 충격파 링 방사 |
| 전환 중 | 보스 주변 에너지 파티클 폭풍 + 카메라 셰이크 |
| 전환 완료 | 발광 소멸 + 새 페이즈 특성 이펙트 활성화 |

### 3.2 체크포인트 활성화 이펙트

- 활성화 시 체크포인트 오브젝트 주변 상향 파티클 버스트
- 활성화 후 상시 발광 루프 (비활성 시 발광 없음)

### 3.3 포탈 / 지역 이동 이펙트

| 상태 | 이펙트 |
| --- | --- |
| 포탈 대기 중 | 포탈 오브젝트 주변 에너지 소용돌이 루프 |
| 지역 이동 시작 | 화면 전체 화이트아웃 또는 에너지 웨이브 효과 |
| 도착 씬 페이드인 | 화면 다크에서 점진적 페이드 |
